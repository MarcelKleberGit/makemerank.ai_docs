## Overview

In MakeMeRank, you first generate **topic suggestions** and then turn one of them into a finished **LinkedIn / B2B post** or a **long-form blog article**. The suggestions are shaped by your report context, persona, and existing content.

## Which contexts can flow in

Content generation can combine several sources:

- **report context** from a Full Report
- a **persona** for tone, language, and perspective
- **Content Sources** as additional content context
- a **LinkedIn connection** for extra enrichment
- **additional inspiration** from a free text field

## Which formats are available

| Format | Use case |
|--------|----------|
| **LinkedIn / B2B post** | A shorter, sharper post for social and personal reach |
| **Long blog article** | A more detailed article with more depth and structure |

## The current flow in the app

<Steps>
  <Step title="Choose scan context">
    By default, MakeMeRank uses the active report. In freestyle flows, you can also work without report context once enough of your own context is available.
  </Step>

  <Step title="Choose a format">
    Select whether you want to create a LinkedIn / B2B post or a long blog article.
  </Step>

  <Step title="Choose a persona">
    A persona is required. It controls tone, language, and the persona-specific context that can be used.
  </Step>

  <Step title="Refine the context">
    You can optionally use LinkedIn, add extra inspiration, and fine-tune how strongly each source should influence the suggestions.
  </Step>

  <Step title="Generate 2 topic suggestions">
    MakeMeRank first creates two topic suggestions. You then choose which one should be expanded.
  </Step>

  <Step title="Generate the draft">
    You can optionally enable a post image and choose an image theme before generating the final output.
  </Step>

  <Step title="Regenerate if needed">
    If you want, you can give the system short feedback and regenerate the draft again.
  </Step>
</Steps>

## Content without an active report

MakeMeRank can also generate topic ideas without a selected report if enough of your own context already exists. This is currently possible when:

- LinkedIn is connected
- or at least **5 saved pieces of content** exist as Content Sources

In that case, MakeMeRank skips report context as an input source and works from your existing context instead.

## Advanced weighting

The advanced weighting control lets you decide which sources should shape your suggestions more strongly. This is useful when you want to lean more heavily on existing content or intentionally allow more new topics.

## Better results with personas and Content Sources

The quality of suggestions improves significantly when both are set up well:

- **Personas** define the style and perspective.
- **Content Sources** provide real material from your existing content.

<Info>
Content generation is tightly connected to the Full Report, but it can now also work more flexibly with your own context. For the strongest results, combine report context, a persona, and Content Sources.
</Info>

If you want to set up those two building blocks first, continue with [Content Sources](/guides/content-sources) and [Personas](/guides/writing-styles).
