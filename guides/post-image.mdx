## Overview

In content generation, you can optionally generate a **post image** alongside the final draft. Image generation is optional and is enabled directly when creating the final output.

## What you can control

When generating a post, you can:

- turn image generation on or off
- choose an existing **image theme**
- create custom image themes on the **Personas** page

## How image themes work

An image theme is a visual style instruction for generated images, for example:

- color direction
- atmosphere
- the overall visual look or rendering style

Custom themes later appear directly in content generation as selectable options.

## What you need for good results

- a selected **scan context** or enough of your own context
- a **persona**
- optionally matching **Content Sources**

For the full setup behind content generation, see [Content Generation](/guides/content-generation) and [Personas](/guides/writing-styles).
